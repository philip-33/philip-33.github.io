Bootstrap Template Analysis

This is a breakdown of the bootstrap template I used for this site.

Layout and Content: <!-- Bootstrap core CSS --> <link href="vendor/bootstrap/css/bootstrap.min.css" rel="stylesheet"> <!-- Custom styles for this template --> <link href="css/heroic-features.css" rel="stylesheet"> These are straightforward. The bootstrap css file is a local minified copy, and the css file for the template itself is also local. <nav class="navbar navbar-expand-lg navbar-dark bg-dark fixed-top"> <div class="container"> <a class="navbar-brand" href="#">Start Bootstrap</a> … … </li> </ul> </div> </div> </nav> This is a navbar that sits at the top of the entire page. It spans the entire width of the page, but the content (Start bootstrap on the left and nav links on the right) matches the container that the rest of the page sits in. The nav links collapse to a hamburger with the same nav links when the width goes below a certain size. <header class="jumbotron my-4"> <h1 class="display-3">A Warm Welcome!</h1> <p class="lead">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ipsa, ipsam, eligendi, in quo sunt possimus non incidunt odit vero aliquid similique quaerat nam nobis illo aspernatur vitae fugiat numquam repellat.</p> <a href="#" class="btn btn-primary btn-lg">Call to action!</a> </header> The header of the page has several important elements. First is the jumbotron, designed for headlines. It also uses a class “my-4” this is for setting margins on the ‘y’ axis to a setting of 4 on a scale of 0-5. These are built into bootstrap and exist for each side as a whole as well as individually, x axis (left and right), and y axis (top and bottom). For the cards, after <div class="row text-center"> Are the cards themselves. These use col-lg-3 so that when the display is “large”, then the cards appear as 4-in-a-row. The col-md-6 class shows that when the display is “medium”, the cards will only show 2-in-a-row. The ‘card’ classes I thought were part of the template are actually part of bootstrap. <div class="col-lg-3 col-md-6 mb-4"> <div class="card"> <img class="card-img-top" src="http://placehold.it/500x325" alt=""> <div class="card-body"> <h4 class="card-title">Card title</h4> <p class="card-text">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Sapiente esse necessitatibus neque.</p> </div> <div class="card-footer"> <a href="#" class="btn btn-primary">Find Out More!</a> </div> </div> </div> The footer uses a padding class just like the margin class above. <footer class="py-5 bg-dark"> <div class="container"> <p class="m-0 text-center text-white">Copyright © Your Website 2017</p> </div> At the end are script tags for the different JS libraries used for the page <script src="vendor/jquery/jquery.min.js"></script> <script src="vendor/popper/popper.min.js"></script> <script src="vendor/bootstrap/js/bootstrap.min.js"></script>

What I need to know to completely understand the template: I need to know what I’m missing. From the looks of it, this template literally only adds the tag structure for the navbar and cards, while everything else is bootstrap.
