Git on your own OS

I already had Git installed on my Linux machine so this was easy to set up. With a portable Linux install it's not too difficult to "borrow" a computer when I'm not at home and boot into a desktop that is mostly ready to use for the class. Since all of my Git experience is through GitHub or the desktop client, I'm not familiar with the command line at all. I found this guide: Git 101 - Quick Start Guide and it seems to provide the basics that allow getting started from the command line. The best feature about it is the explanations for each command that aren't overly detailed, and the commands are grouped logically. The main drawback is that it doesn't have context for the commands, so it's not exactly clear when they should be used or what to do when things go wrong.
