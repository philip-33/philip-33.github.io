GitHub Topics

Of all the tools used today as part of any realm of development, GitHub is probably one of the most ubiquitous tools in use. This is a form of system that performs version control, which allows for multiple people to easily contribute to a project with varying levels of authority and control as well as keeping a log of all changes to a project. This log goes all the way back to the moment it was added to the repository.

There are other version control systems, but GitHub is free to use for public projects and the differences between it and other similar systems are often relatively minor. The topics presented here are a good core to begin working with GitHub and allow for working with branches and local datasets. A common complaint from companies that hire Computer Science graduates is that these students never learned how to use GitHub.
