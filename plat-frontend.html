Front-End Frameworks

Milligram was the framework that seemed most interesting to me. The homepage (milligram.io) was missing some text when I first looked at it:

first view of the page

After looking around, it turned out that one of my plugins was blocking a script that was being loaded over http instead of https. This framework was the most interesting to me because while all the frameworks were meant to look good on mobile and desktop, Milligram is small. Milligram uses CSS Flexbox which means that in addition to the benefits of flexbox features (easy centering/formatting of elements on a page, among other things) it also doesn't require additional script files to load for this to work. This means a smaller total page size. It uses google fonts, Roboto by default, so there's a good and growing chance that the client will already have the font cached. One drawback is that it will likely not work as well in older browsers.
