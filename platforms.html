Platforms

The topics presented here are limited, but deal with some absolute basics that a professional developer will be expected to understand. Perhaps the most important is the first topic of web hosting. This represents the base that is necessary for any web development to be put to use. There is an incredible range of hosting available, and it is necessary to understand hosting and to know what kind of hosting is necessary, best, or overkill for a given project.

Wordpress and how it is used for ecommerce is another fundamental component for developers. There are other platforms besides Wordpress but the requirements of selling, charging, and shipping online will need to be met regardless of the technology.

The last topic here deals with frameworks. These are sets of resources that are available for developers to use to add specific and/or unique functionality and style to their projects. For this particular class we used bootstrap and a template specifically for it, but there are many options available.
