Chrome DevTools

I have a bit of familiarity with DevTools from troubleshooting websites before, but I didn't know too much about it beyond CSS hierarchy wrangling. The timeline feature was useful to see how long things take to load, but for a lot of the DevTools features I didn't really know what I was doing.

Prashant Palikhe: Chrome DevTools Masterclass

I was looking for videos that talked specifically about the timeline, so when I first saw it I thought it would be beyond. After watching a few minutes though, I realized it was in the sweet spot of challenging but not too challenging. For a large chunk of the video he's debugging load times on a web app (running on a virtualized phone, but whatever), which is exactly what I want to know more about.

This video is great because it's not just about DevTools, it's about debugging an app and making it better using DevTools, which is exactly what I want to know how to do. He talks about improving load times and UX and how certain things should look in DevTools when the app is working well.
