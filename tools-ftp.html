FTP

FTP is a basic protocol for transferring files, but does not have a way to do this securely. So, instead of talking about FTP, here's some info on SFTP (SSH File Transfer Protocol), which is a little different.

SSH FTP info @ Indiana University

This is a link from Indiana State that has a good overview of the protocol and related commands, as well as recommending clients to use with it. It's all presented in the context of connecting to their servers (don't do that), but the overview is still useful and explains the basics of using the protocol on their system.

SSH File Transfer Protocol on Wikipedia

Wikipedia provides a little more detail of the protocol and its history.

Links aside, the main benefit of the protocol is that the connections are negotiated securely, so that any data in transit is encrypted. It is also possible to use SSH keys instead of passwords to connect to SFTP servers.
